### **Desafio 1 – Divisão por zero**

**Problema:**
Crie um programa que divida dois números inteiros fornecidos pelo usuário:

```java
int a = 10;
int b = 0;
```

* Use `try/catch` para capturar a **exceção de divisão por zero** e imprimir `"Não é possível dividir por zero"`.

**Pergunta lógica:** Qual exceção Java é lançada quando se tenta dividir por zero?

### **Desafio 2 – Conversão de tipos**

**Problema:**
Tente converter uma string para número inteiro:

```java
String valor = "abc";
```

* Use `try/catch` para capturar a **exceção de número inválido** e imprimir `"Formato inválido"`.

**Pergunta lógica:** Qual exceção Java é lançada quando a conversão de string para número falha?

### **Desafio 3 – Lançando exceções**

**Problema:**
Crie um método `verificarIdade(int idade)` que:

* Lança uma exceção `IllegalArgumentException` se a idade for menor que 18
* Caso contrário, imprime `"Idade válida"`

**Pergunta lógica:** Como lançar exceções personalizadas para validar condições de negócio?
