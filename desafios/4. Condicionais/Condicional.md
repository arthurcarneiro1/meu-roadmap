### **Desafio 1 – Verificação de maioridade**

Você tem:

```java
int idade = 18;
```

**Problema:**
Crie uma estrutura condicional que imprima:

* `"Maior de idade"` se `idade >= 18`
* `"Menor de idade"` caso contrário

**Pergunta lógica:** Qual estrutura condicional simples atende a esse caso?

### **Desafio 2 – Classificação de números**

Você tem:

```java
int numero = -5;
```

**Problema:**
Crie uma condicional que imprima:

* `"Positivo"` se o número for maior que 0
* `"Negativo"` se o número for menor que 0
* `"Zero"` se o número for igual a 0

**Pergunta lógica:** Qual estrutura permite testar **mais de duas condições**?

### **Desafio 3 – Verificação de múltiplos**

Você tem:

```java
int valor = 15;
```

**Problema:**
Crie uma condicional que imprima:

* `"Múltiplo de 3 e 5"` se `valor` for múltiplo de 3 **e** de 5
* `"Múltiplo de 3"` se for múltiplo apenas de 3
* `"Múltiplo de 5"` se for múltiplo apenas de 5
* `"Não é múltiplo de 3 ou 5"` caso contrário

**Pergunta lógica:** Como combinar operadores lógicos (`&&`, `||`) com o operador de módulo `%` para várias verificações?
