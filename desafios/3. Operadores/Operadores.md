### **Desafio 1 – Operadores aritméticos**

Você tem:

```java
int a = 10;
int b = 3;
```

**Problema:**
Crie uma expressão que calcule o **resto da divisão de `a` por `b`** e armazene em uma variável chamada `resto`.

**Pergunta lógica:** Qual operador você deve usar para obter o resto?

---

### **Desafio 2 – Operadores lógicos**

Você tem duas variáveis booleanas:

```java
boolean ligado = true;
boolean funcionando = false;
```

**Problema:**
Crie um boolean `status` que seja `true` **apenas se ligado for verdadeiro e funcionando também for verdadeiro**.

**Pergunta lógica:** Qual operador lógico garante que **ambos** precisam ser verdadeiros?

---

### **Desafio 3 – Operadores de incremento e atribuição**

Você tem:

```java
int x = 5;
```

**Problema:**

1. Aumente `x` em 1 usando **incremento pós-fixado**.
2. Depois multiplique `x` por 3 usando **atribuição combinada**.

**Pergunta lógica:** Qual a diferença entre `x++` e `++x`? E como operadores combinados simplificam a escrita?
