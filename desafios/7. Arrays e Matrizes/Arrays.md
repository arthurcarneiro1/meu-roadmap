### **Desafio 1 – Acesso a elementos**

Você tem o array:

```java
int[] numeros = {10, 20, 30, 40, 50};
```

**Problema:**
Imprima o **primeiro** e o **último** elemento do array.

**Pergunta lógica:** Como acessar elementos específicos de um array usando índices?

---

### **Desafio 2 – Soma dos elementos**

Você tem o array:

```java
int[] numeros = {1, 2, 3, 4, 5};
```

**Problema:**
Calcule a **soma de todos os elementos** do array usando um loop `for`.

**Pergunta lógica:** Como percorrer todos os elementos de um array e acumular valores?

---

### **Desafio 3 – Maior valor**

Você tem o array:

```java
int[] numeros = {12, 45, 7, 23, 89, 34};
```

**Problema:**
Encontre o **maior valor** do array e imprima-o.

**Pergunta lógica:** Como percorrer o array e comparar elementos para encontrar o maior número?
