### **Desafio 1 – Conexão com banco**

**Problema:**

* Crie um programa Java que se conecte a um banco de dados **MySQL** usando JDBC.
* Use URL, usuário e senha de teste.
* Imprima `"Conexão bem-sucedida"` se a conexão for realizada.

**Pergunta lógica:** Qual driver e URL são necessários para conectar um banco MySQL usando JDBC?

---

### **Desafio 2 – Executando consulta**

**Problema:**

* Usando a conexão JDBC, execute uma consulta `SELECT * FROM clientes`.
* Percorra o `ResultSet` e imprima `id` e `nome` de cada cliente.

**Pergunta lógica:** Como usar `Statement` e `ResultSet` para executar consultas e ler resultados?

---

### **Desafio 3 – Inserção com PreparedStatement**

**Problema:**

* Crie um método que insira um novo cliente (`id`, `nome`) na tabela `clientes` usando **PreparedStatement**.
* Execute o método para adicionar pelo menos um cliente.

**Pergunta lógica:** Por que é importante usar `PreparedStatement` ao invés de concatenar strings em queries SQL?
